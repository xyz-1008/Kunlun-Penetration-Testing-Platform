## 快速入门

### 启动应用

双击 `AutoPenTest.exe` 即可启动应用，无需安装。

### 界面布局

应用启动后，您将看到以下界面元素：

- **顶部菜单栏**: 包含文件、视图、工具、帮助等菜单
- **模块标签栏**: 位于界面顶部，可切换不同功能模块
- **主工作区**: 显示当前选中模块的内容
- **状态栏**: 位于界面底部，显示内存、磁盘使用情况和就绪状态

### 添加测试目标

1. 点击 **Target** 标签页
2. 点击 "添加目标" 按钮
3. 输入目标URL或IP地址
4. 配置目标范围
5. 点击 "保存"

## 核心功能使用

### Proxy 代理拦截

Proxy 模块用于拦截和修改 HTTP/HTTPS 请求。

#### 配置代理

1. 打开 **Proxy** 标签页
2. 点击 "选项" 或 "设置"
3. 配置监听地址和端口（默认 127.0.0.1:8080）
4. 点击 "启动" 开始监听

#### 浏览器配置

在浏览器中配置代理设置：

- **代理地址**: 127.0.0.1
- **端口**: 8080
- **类型**: HTTP/HTTPS

#### 拦截请求

1. 确保 "拦截" 开关已打开
2. 在浏览器中访问目标网站
3. 请求将显示在 Proxy 拦截窗口中
4. 修改请求参数后点击 "转发"

### Intruder 暴力破解

Intruder 模块用于字典攻击和参数模糊测试。

#### 基本使用

1. 打开 **Intruder** 标签页
2. 设置目标请求
3. 选择攻击类型：
   - **Sniper**: 单参数攻击
   - **Battering Ram**: 多参数相同值攻击
   - **Pitchfork**: 多参数并行攻击
   - **Cluster Bomb**: 多参数组合攻击
4. 配置 Payload 字典
5. 点击 "开始攻击"

#### 结果分析

攻击完成后，查看结果列表：

- 按响应长度排序，发现异常响应
- 按状态码排序，发现不同状态码
- 查看响应内容，寻找敏感信息

### Repeater 请求重放

Repeater 模块用于手动修改和重放请求。

#### 使用方法

1. 从 Proxy 拦截中右键选择 "发送到 Repeater"
2. 或直接打开 **Repeater** 标签页
3. 修改请求参数
4. 点击 "发送" 查看响应
5. 可保存多个请求进行对比

### Decoder 编解码

Decoder 模块用于编码、解码和哈希计算。

#### 支持的操作

- **URL 编码/解码**
- **Base64 编码/解码**
- **HTML 实体编码/解码**
- **Hex 编码/解码**
- **MD5/SHA1/SHA256 哈希计算**

#### 使用方法

1. 打开 **Decoder** 标签页
2. 在输入框中输入或粘贴文本
3. 选择编码/解码操作
4. 查看输出结果

## 常见问题

### Q: 应用无法启动

**A**: 请检查以下几点：
1. 操作系统是否为 Windows 10/11 64位
2. 是否有杀毒软件阻止了应用运行
3. 尝试右键选择 "以管理员身份运行"

### Q: Proxy 无法拦截请求

**A**: 请检查：
1. 浏览器代理设置是否正确配置
2. Proxy 监听端口是否为 8080
3. 拦截开关是否已打开

### Q: HTTPS 网站无法访问

**A**: 需要安装应用的 CA 证书：
1. 在 Proxy 设置中导出 CA 证书
2. 在浏览器中导入并信任该证书

### Q: 如何保存测试数据

**A**: 使用 "文件" 菜单中的 "保存项目" 功能，可将当前测试状态保存到本地文件。

## 高级技巧

### 批量导入目标

1. 准备一个文本文件，每行一个目标URL
2. 使用 "文件" > "导入" > "导入目标" 功能
3. 选择文件并确认导入

### 自定义 Payload 字典

1. 在 Intruder 中点击 "Payloads" 标签
2. 选择 "加载外部文件"
3. 选择自定义字典文件

### 使用插件扩展功能

1. 打开 **插件商店** 标签页
2. 浏览可用插件
3. 点击 "安装" 添加新功能

### 导出测试报告

1. 完成测试后打开 **报告** 标签页
2. 选择报告模板
3. 点击 "导出" 生成 HTML/PDF 报告
